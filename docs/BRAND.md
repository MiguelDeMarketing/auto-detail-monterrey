# Auto Detail Monterrey — Brand Guidelines

## Colores
- Primary: #1a1a2e (Navy oscuro)
- Accent: #e94560 (Rojo cereza)
- Surface: #f5f5f5 (Gris claro)
- Gold: #d4af37 (Dorado premium)
- Nota: colores pendientes de formalizacion — extraidos del sitio actual

## Tipografia
- Pendiente de definir formalmente
- Sitio actual usa fuentes del sistema

## Tono
Premium pero accesible. Enfoque en calidad de materiales y resultados visuales. Mostrar el antes/despues. Vocabulario: ceramico, PPF, vinyl wrap, polarizado, nano-ceramico.

## Anti-branding
- NO lenguaje de taller mecanico — es detailing premium
- NO precios en la web — comunicar valor, cotizar personalizado
